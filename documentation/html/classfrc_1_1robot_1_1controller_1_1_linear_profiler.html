◆ LinearProfiler() [4/4]

frc.robot.controller.LinearProfiler.LinearProfiler ( )
inline

Creates a LinearProfiler. The default max velocity and max acceleration are both positive infinity (this implies a profile which constantly accelerates until its reached half its target, then constantly deccelerates). The default PID constants are 1, 0, and 0 respectively. The default period is 0.02 seconds
